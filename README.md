## Project Summary
We have chosen case study 7, AI in Financial Literacy as our project's focus. We have build a prototype of mobile application call FinWise AI using Figma. FinWise AI is an intelligent financial companion designed to bridge the critical financial literacy and protection gap across the ASEAN region. FinWise AI solves this by transforming passive expense tracking into actionable wealth management.
